Chave API do Gemini

Siga os passos abaixo para gerar sua chave gratuitamente e habilitar a transcrição.

1

Acesse o Google AI Studio

Vá até o site oficial do Google AI Studio e faça login com sua conta do Google.

2

Encontre a área de API

No menu lateral esquerdo, procure pelo botão chamado "Get API key" ou "Obter chave de API" e clique nele.

3

Crie sua Chave

Clique no botão azul "Create API key". Selecione a opção para criar em um novo projeto. O Google vai gerar um código longo (ex: AIzaSyB...).

4

Copie e Cole

Copie esse código, volte para a aba "Gerar Audiobook" neste site, tente enviar um arquivo e cole sua chave no pop-up que aparecer.
